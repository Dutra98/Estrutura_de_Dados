2-
a) Teste e explique a diferença, caso exista, entre
p++; (*p)++; *(++p);

R:
p++ -> O ponteiro anda para o próximo endereço de inteiro.
(*p)++ -> Aumenta o valor para qual p está apontando.
*(++p) -> O ponteiro anda para o próximo endereço de inteiro e depois acessa o valor.

b) O que quer dizer *(p+10)?

R: Acessar o valor que está 10 posições à frente do endereço apontado por p.

parâmetros recebidos. Sua assinatura deve ser:void troca(float *a, float *b);

#include <stdlib.h>
#include <stdio.h>

void troca(float *a, float *b) {
    float temp;

    temp = *a;
    *a = *b;
    *b = temp;
}

int main() {

    float x = 5.5;
    float y = 9.2;

    troca(&x, &y);

    printf("x = %.1f, y = %.1f\n", x, y);
